# Introduction

## What's this?

`SuperAlertController` is an `UIAlertController` extension.

## Requirements

* iOS 9.0+
* Xcode 9 with Swift 4

## Installation

#### CocoaPods

```ruby
pod 'SuperAlertController'
```
